## Используемые инструменты:
1) **Intellij IDEA** - программа для разработки на языках Java и Kotlin;
2) **Gradle** - система для автоматизации сборки;
3) **JUnit** - фреймворк для языка программирования Java, предназначенный для автоматического тестирования программ;
4) **Selenide** - фреймворк для автоматизированного тестирования веб-приложений;
5) **Браузер** - инструмент для просмотра сайтов, в т.ч. с помощью DevTools;
6) **GitHub** - инструмент для хранения и обмена данными с коллегами/ заинтересованными лицами;
7) **MySQL** или соответствующая СУБД, в случае если потребуется доступ к базе данных;
8) **DBeaver** или соответствующая программа для управления и работы с БД;
9) **Lombok** библиотека для генерации стандартных методов, в т.ч. - геттер/сеттер, конструкторы и т.д.
10) **FAKER** библиотека для генерации случайных данных, в т.ч. имя, телефон, e-mail.
